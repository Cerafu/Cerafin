Texel Parameter Optimizer

An automated optimization network built to tune raw evaluation terms. It processes localized self-play game datasets, applies custom mathematical Sigmoid algorithms, and calculates global Mean Squared Error deviations to iteratively fine-tune internal piece-square matrices.

Zobrist Transposition Core

A computational memory architecture designed to minimize exponential alpha-beta tree growth. Implements highly optimized 64-bit integer hashes for positions, multi-threaded popcounts, dynamic depth priority selection strategies, and bitwise indexing logic.
